Проблема здесь такая - изначально доступно только 3 языка, поддержку которых я реализовал.
Чтобы добавить 4-й язык, нужно доработать 2 сервиса на бэкенде и фронтенд.

Дипломным руководителем было предложено решить эту проблему с помощью плагинизации.
Тем более дипломный проект магистра должен предусматривать расширяемость системы без модификации ядра.

# Способы подгрузки плагинов

Java ServiceLoader пример подгрузки классов.
Нужно положить в classpath и указать название сервиса.

JS dynamic import пример подгрузки модулей.
Указываем ссылку до скрипта и динамически подгружаем.
