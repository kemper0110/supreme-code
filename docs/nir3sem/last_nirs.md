## Варианты передачи параметров и проверки результатов:

**STDIO**

программа сама вычитывает параметры и печатает результат.

**Вызов функции**

Пользователь реализует функцию.

```cpp
vector<int> twoSum(vector<int>& nums, int target){}
```

И пишет свои тестовые данные в текстовом формате.

```
nums = [2,7,11,15]
target = 9
```

Какие-то программные прослойки прокидывают эти параметры в функцию и вызывают.

**Unit-тесты**

Пользователь реализует функцию.
Тесты используют эту функцию и проверяют результат.

## Контейнеризация

Изучал как обезопасить выполнение недоверенного кода.

```shell
docker run -it --rm \
  --storage-opt size=128M \
  --device-read-iops /dev/sda:1000 \
  --device-write-iops /dev/sda:1000 \
  --read-only \
  --tmpfs /usr/app:size=32M \
  --memory 128M \
  --cpus 1.0 \
  --ulimit nproc=64 --ulimit nofile=512 \
  --network=none \ 
  gcc:13.2.0
```
